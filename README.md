## Instalación

Instala mi proyecto con NPM!

```bash
  Terminal Back-end
  cd api
  npm install
  npm run dev

  Terminal Front-end
  cd client
  npm install
  npm run dev
  
```
## Environment Variables

Para ejecutar el proyecto, necesitas añadir las variables de entorno correspondientes 

Api

`MONGO_URI`

`JWT_KEY`

Client

`clientId`
